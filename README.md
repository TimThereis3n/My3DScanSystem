本文件为SZTU三维感知技术实验室暑期进行的结构光三维重建项目中Basler相机与DLP4500投影仪与GCD平移台实现同步的代码文件。同步时序精度达毫秒级（优化后精度可大幅提高）。 文件主体上分为4个模块：投影仪类模块，相机模块，位移台模块，主控制流程函数。三个模块都封装了一个类，很容易调用，理论上来讲可以直接复制到你的项目里使用。投影仪部分要结合使用DLP4500生产厂商提供的LightCrafter4500软件来设置DLP4500投影仪的图片烧录，播放序列以及投影曝光长度等一系列参数（这玩意很难集成到程序里）。 GCD平移台为大恒光电公司旗下的一款平移台系列产品，可以上官网看他的数据手册，是用串口USART进行通信的，但是它的通信协议是自定义的二进制帧格式，具体可参考platform.cpp文件内。
